# Relume Component Library

HTML + Tailwind CSS components extracted from the Relume Figma Kit v3.7.

## Structure

```
components/
  hero/
  navbar/
  cta/
  features/
  ...
```

Each component is a standalone `.html` file with Tailwind CSS classes.

## index.json

AI-browsable manifest with component metadata (name, category, description, path).
